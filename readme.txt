9. Working with video, images, IO streams and painting
You have AVI file (noncompressed, 24 bit RGB).
Write a simple application with GUI that reads first 3 frames from AVI (attached to this test). Choose file with the standard file 
chooser. Application paints these 3 frames in panel near each other horizontally with opacity 100, 80 and 60 percent. Application should 
be packaged as a JAR file and could be run as 'java -jar app.jar <args>'. Application should be written 100% in Java.
